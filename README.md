# visdata
visdata is a lightweight, intuitive web-based visualization tool designed to transform raw data into insightful, interactive graphs. Whether you're working with spreadsheets or developer-friendly formats, visdata makes data storytelling effortless.

## Key Features

- Multi-Format Support: Seamlessly upload and parse .xls, .xlsx, CSV, and JSON files.
- Dynamic Visuals: Switch between various graph types to find the best representation for your data.
- Local-First: Keep your data secure by processing and visualizing it right in your browser.

## Supported File Types
| Format | Extension | Description |
| ------ | --------- | ----------- |   
| Excel | .xls, .xlsx | Standard spreadsheet files. |
| CSV | .csv | Comma-separated values for large datasets. |
| JSON | .json | Structured data objects for developers. |

## Quick Start Guide

### Prerequisites
Before you begin, ensure you have the following installed:

- Node.js (LTS recommended)
- npm (comes bundled with Node.js)

### Installation
Get your local environment up and running in minutes:

1. Clone the Repository
```bash
git clone https://github.com/SauRavRwT/vidata.git
cd vidata
```
2. Install Dependencies
```bash
npm install
```
3. Launch the App
```Bash
npm start
```

The application will automatically open at `http://localhost:3000.`

## Available Scripts
